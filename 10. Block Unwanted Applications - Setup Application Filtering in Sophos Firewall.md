# ✅ How to Block Unwanted Applications

## Setup Application Filtering in Sophos Firewall

***

## 1️⃣ Why Is Application Filtering Important?

**Application Filtering (App Control)** allows you to **identify, allow, limit, or block applications**, regardless of:

*   Port number
*   Protocol
*   Encryption

### 🔐 Why It Matters in Real Organizations

✅ Prevents productivity loss  
✅ Reduces security risks  
✅ Controls bandwidth usage  
✅ Enforces company IT policies  
✅ Blocks risky or non‑business apps

***

### ✅ Real‑World Examples

| Department    | Why App Filtering Is Needed          |
| ------------- | ------------------------------------ |
| HR            | Block games, file sharing, streaming |
| Finance       | Block cloud storage & unknown apps   |
| IT            | Allow tools, restrict entertainment  |
| Guest Network | Allow internet only                  |

📌 **Without application filtering**, users can bypass basic firewall rules using HTTPS or custom ports.

***

## 2️⃣ What Is Application Filtering in Sophos Firewall?

Sophos uses **App Control (Layer‑7 inspection)** to detect applications such as:

*   File sharing tools
*   Streaming media
*   Games
*   Social media
*   Cloud applications
*   Unknown & risky apps

Even if they run over **HTTPS (443)**, Sophos can detect them.

***

## 3️⃣ Create an Application Filter Policy

### Navigation Path

    Protect → Applications → Application Filter → Add

***

### ✅ Create New Policy

    Name:      Block_HR_Application
    Template:  Block generally unwanted apps

Click **Save**

📌 This template initially blocks common risky applications.

***

## 4️⃣ Customize the Application Filter Policy

### Edit the Policy

    Protect → Applications → Application Filter
    Search: Block_HR_Application → Edit

***

### ✅ Why Remove Default Blocked Apps?

In **large organizations**, default templates may:

*   Block required applications
*   Cause business disruption
*   Be too generic

✅ **Best Practice**:

> Remove default entries and **create a clean, customized policy**

***

### Step 1: Remove Default Applications

*   Delete all default blocked applications
*   Start with an empty policy

***

### Step 2: Add Custom Blocked Applications

Click **Add**

#### Select by Category

Add the following categories:

✅ **File Transfer**  
✅ **Gaming**  
✅ **Streaming Media**

***

### Configuration Settings

    Selection:   Select All
    Schedule:    All the time
    Action:      Deny

Click **Save**

✅ Your custom HR application‑blocking policy is ready

***

## 6️⃣ What Are Cloud Applications? (Important Concept)

### Navigation

    Protect → Applications → Cloud Applications

***

### What Is a Cloud Application?

Cloud applications are **web‑based services** like:

*   Google Drive
*   Dropbox
*   OneDrive
*   Salesforce
*   Zoom
*   Microsoft 365

Sophos identifies:

*   Which cloud apps are being used
*   By whom
*   How frequently

***

### ✅ Real‑World Use Case

> IT team reviews cloud app usage and decides:

*   ✅ Allow Microsoft 365
*   ❌ Block personal Google Drive
*   ❌ Block unauthorized cloud storage

📌 This helps prevent **data leakage**.

***

## 7️⃣ Important Rule: App Filtering Alone Is Not Enough

❗ Application filtering **does nothing** unless it is **attached to a firewall rule**.

***

## 8️⃣ Attach Application Filter to Firewall Rule

### Navigation

    Protect → Firewall Rules

Open:

    HR_Rule_Policy → Edit

(or your LAN → WAN rule for HR users)

***

### Under **Other Security Features**

Enable:

    Identify and control application (App Control)

Select:

    Block_HR_Application

Click **Save**

✅ Now the policy is active and enforced

***

## 9️⃣ Traffic Flow (Simple View)

    HR User
       ↓
    Firewall Rule (HR_Rule_Policy)
       ↓
    Application Control (Block_HR_Application)
       ↓
    Allowed ❌ / Blocked ✅

***

## 🔁 Example: User Accessing YouTube

| Step               | Result                       |
| ------------------ | ---------------------------- |
| User opens YouTube | Detected as Streaming Media  |
| App Control        | Blocked                      |
| User sees          | Access denied                |
| Log entry          | Recorded in App Control logs |

***

## ✅ Best Practices (Enterprise Standard)

✅ Use **department‑based policies**  
✅ Keep policies **clean & minimal**  
✅ Review **Cloud Application usage regularly**  
✅ Always **attach App Control to firewall rules**  
✅ Enable logging during initial deployment

❌ Avoid:

*   One policy for all departments
*   Blocking without business validation
*   Forgetting to apply policy to rules

***

## ✅ One‑Line Summary

> **Application filtering in Sophos Firewall allows organizations to block unwanted apps like gaming, file sharing, and streaming, ensuring security, productivity, and bandwidth control when properly attached to firewall rules.**
