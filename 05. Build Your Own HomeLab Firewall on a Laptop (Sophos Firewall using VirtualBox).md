# 🔐 Build Your Own HomeLab Firewall on a Laptop (Sophos Firewall using VirtualBox)

This guide explains how to **build a Sophos Firewall HomeLab** on a laptop using **Oracle VirtualBox**.

***

## ✅ Requirements

*   Laptop / PC (Virtualization enabled)
*   Internet connection
*   Oracle VirtualBox
*   Sophos Firewall Virtual Image

***

## 1️⃣ Download & Install Oracle VirtualBox

### 🔗 Download Link

    https://www.virtualbox.org/wiki/Downloads

### Steps

1.  Download **VirtualBox for Windows hosts**
2.  Install VirtualBox with default settings
3.  Reboot the system if prompted

✅ VirtualBox installation completed

***

## 2️⃣ Download Sophos Firewall Virtual Image

### 🔗 Download Link

    https://www.sophos.com/en-us/support/downloads/firewall-installers

### Steps

1.  Scroll to **Virtual Installers**
2.  Select **Firewall OS for VMware**
3.  Download the file
4.  Extract the downloaded archive

📌 After extraction, note the file:

    sf_virtual  (approx. 7 KB)

***

## 3️⃣ Import Sophos Firewall into VirtualBox

### Steps

1.  Open **Oracle VirtualBox Manager**
2.  Click **File → Import Appliance** (or double‑click the file)
3.  Select `sf_virtual`
4.  Click **Next → Finish**

✅ Virtual machine setup will begin automatically

***

## 4️⃣ Start the Sophos Firewall VM

1.  Select the Sophos Firewall VM
2.  Click **Start**

⏳ Firewall will begin booting

***

## 7️⃣ Configure Laptop Network Settings (Windows)

### Open Network Connections

1.  Press **Win + R**
2.  Type:

<!---->

    ncpa.cpl

3.  Press **Enter**

***

### Configure Ethernet Adapter

1.  Right‑click **Ethernet → Properties**
2.  Select **Internet Protocol Version 4 (TCP/IPv4)**
3.  Click **Properties**
4.  Select **Use the following IP address**

#### IP Configuration

    IP Address:      172.16.16.1
    Subnet Mask:     255.255.0.0
    Default Gateway: 172.16.16.16

#### DNS Configuration

    Preferred DNS:   8.8.8.8
    Alternate DNS:   8.8.4.4

5.  Click **OK → OK**
6.  When prompted, click **Yes** to enable network access

✅ Network connection established
