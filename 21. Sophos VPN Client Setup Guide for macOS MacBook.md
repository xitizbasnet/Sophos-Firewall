# Sophos VPN Client Setup Guide for macOS (MacBook)

## 📘 Overview

This document explains how to download, configure, and connect the **Sophos Connect VPN Client** on a **macOS (MacBook)** device using an IPsec remote access VPN configuration.

---

# 🔐 Prerequisites

Before starting, ensure you have:

* Access to the Sophos Firewall
* Valid VPN user credentials
* macOS device with installation permissions
* WinRAR or archive extraction utility installed

---

# 🌐 Access Sophos Firewall

Open your browser and go to the Sophos Firewall portal:

```text
172.16.16.16:444
```

## 🔑 Login Credentials

Enter the user credentials:

```text
Username: admin
Password: ****
```

Click **Login**.

---

# ⚙️ Download Sophos VPN Client and Configuration File

Navigate to:

```text
Configure > VPN > IPsec (Remote Access)
```

## 📥 Download the VPN Client

At the bottom of the page:

* Locate **Download Client**
* Download the client package

## 📥 Export the VPN Configuration File

At the bottom of the same page:

* Locate **Export Connection**
* Download the configuration file

---

# 📂 Downloaded Files

After downloading, you will have the following files:

```text
sezconnectclient.tar
sophosconnect_installer
```

---

# 🗂️ Extract the Configuration File

## Step 1: Open the `.tar` File

Open the following file using **WinRAR**:

```text
sezconnectclient.tar
```

Extract the file.

## Step 2: Locate the Configuration File

After extraction, navigate to:

```text
sezconnectclient > tmp > sezconnectclient.tgb
```

The file size is approximately:

```text
3 KB
```

> 📌 This `.tgb` file is used as the VPN configuration file for macOS.

---

# 🗂️ Extract the Sophos Installer File

## Step 1: Open the Installer Archive

Open the following file using **WinRAR**:

```text
sophosconnect_installer
```

Extract the file.

## Step 2: Locate the macOS Installer Package

Inside the extracted folder, search for:

```text
Sophos Connect_1.4_(IPsec).pkg
```

Approximate file size:

```text
6989 KB
```

---

# 💡 Alternative Method to Download the Installer

> 📌 The `Sophos Connect_1.4_(IPsec).pkg` installer can also be downloaded from the Sophos User Portal.

## 🌐 Access the User Portal

Open:

```text
172.16.16.16:443
```

## 🔑 Login Credentials

Enter the user credentials:

```text
Username: xitiz.basnet
Password: ****
```

Click **Login**.

---

# 🔍 Download Installer from User Portal

Navigate to:

```text
VPN
```

Under:

```text
Sophos Connect Client
```

Search for:

```text
Download for macOS
```

Download the installer.

---

# ⚠️ Important Note

> The user portal allows users to download **only the installer file**.

> The **configuration file (`.tgb`) cannot be downloaded from the user portal**.

> To obtain the configuration file, the user must contact the **Sophos Administrator**.

---

# 🖥️ Install Sophos Connect on macOS

Now let’s continue with the Sophos installer file.

## Step 1: Locate the Installer

Go to the **Downloads** folder and locate:

```text
Sophos Connect_1.4_(IPsec).pkg
```

## Step 2: Install the Application

Double-click the installer and follow the steps:

```text
Continue > Install
```

---

# 🔐 macOS Permission Prompts

The installer will request the laptop password.

Enter:

```text
Laptop Password: *****
```

Then proceed with:

```text
Install Software
```

You may receive additional macOS permission prompts:

* Click **OK** to allow access to the Desktop folder
* Click **OK** to allow control system events

After completion:

```text
Installation was successful
```

Click:

```text
Close
```

---

# 🌐 Import VPN Configuration

After installation:

* Locate the **Sophos icon** in the top navigation/menu bar

## Import Connection

Click:

```text
Import Connection
```

Select the configuration file:

```text
sezconnectclient.tgb
```

Then click:

```text
Open
```

Click on the imported connection to run it.

---

# 🔑 Connect to Sophos VPN

Enter the VPN login credentials:

```text
Username: xitiz.basnet
Password: ******
```

Enable:

```text
✔ Save username and password
```

Click:

```text
Sign In
```

---

# ✅ Connection Successful

The Sophos VPN has now been connected successfully.

---

# 📌 Summary

## Files Required

| File Name                        | Purpose                     |
| -------------------------------- | --------------------------- |
| `sezconnectclient.tgb`           | VPN Configuration File      |
| `Sophos Connect_1.4_(IPsec).pkg` | Sophos VPN Client Installer |

---

# ⚠️ Notes

* The `.tgb` configuration file is mandatory for VPN connection setup.
* The installer can be downloaded from either:

  * Sophos Firewall Admin Portal
  * Sophos User Portal
* The configuration file must be provided by the Sophos Administrator.
* macOS administrator privileges are required during installation.

---

# 🛠️ Troubleshooting Tips

## If the VPN Does Not Connect

* Verify the username and password
* Ensure the `.tgb` configuration file is imported correctly
* Restart the Sophos Connect application
* Confirm internet connectivity
* Contact the Sophos Administrator if the issue persists
