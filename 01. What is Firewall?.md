# 🛡️ What Is a Firewall?
## *Sophos Firewall – Beginner Guide*

---

## 📘 Overview

A **firewall** is a **network security device or software** that **monitors, controls, and filters network traffic** based on predefined security rules.  
Its primary purpose is to **protect internal networks from unauthorized access, cyber threats, and malicious traffic**, while still allowing legitimate communication.

---

## ❓ Why a Firewall Is Needed

Without a firewall, your network is directly exposed to:

- 🚨 Hackers and unauthorized access  
- 🦠 Malware, ransomware, and viruses  
- 📤 Data theft and network attacks  

A firewall acts like a **security gate** between:

- 🔒 **Trusted network** (LAN / internal users)  
- 🌐 **Untrusted network** (Internet)

---

## ⚙️ How a Firewall Works (Simple View)

1. **Traffic enters the firewall**
2. The firewall checks:
   - Source & destination IP
   - Port numbers
   - Protocols (TCP, UDP, ICMP)
   - Security rules
3. The firewall then decides to:
   - ✅ **Allow**
   - ❌ **Block**
   - 🔍 **Inspect deeply**

---

## 🧱 Types of Firewalls

### 1️⃣ Packet-Filtering Firewall
- Checks basic information (IP address, port, protocol)
- Fast performance but provides limited security

### 2️⃣ Stateful Firewall
- Tracks active and established connections
- More secure than basic packet filtering

### 3️⃣ Next-Generation Firewall (NGFW)
- Performs deep packet inspection
- Provides application control
- Includes intrusion prevention
- Offers web and malware protection  

✅ **Sophos Firewall belongs to this category**
