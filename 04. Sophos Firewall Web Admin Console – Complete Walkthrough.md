# ✅ Sophos Firewall Web Admin Console – Complete Walkthrough

The **Sophos Firewall Web Admin Console** is a **browser‑based management interface** used to configure, monitor, and manage all firewall functions.

***

## 1️⃣ Accessing the Sophos Web Admin Console

### Default Access Details

*   **URL:**
        https://<Firewall-IP>:4444
    or
        https://172.16.16.16:4444

*   **Default Username:** `admin`

*   **Password:** Set during initial setup

⚠️ Always use **HTTPS** (port 4444).

***

## 2️⃣ Login Screen Overview

When you log in, you will see:

*   Username & password fields
*   Locale selection (if enabled)
*   Warning message for certificate (self‑signed)

✅ After login → **Dashboard appears**

***

## 3️⃣ Web Admin Console Layout Overview

The console is divided into **three main parts**:

### 🔹 Top Bar

*   Logged‑in admin user
*   Logout option
*   Help & Support
*   System notifications

### 🔹 Left Navigation Menu (Main Configuration Area)

*   CONTROL
*   PROTECT
*   DETECT
*   CONFIGURE
*   SYSTEM
*   REPORTS

### 🔹 Main Dashboard Area

*   Real‑time network & security status

***

## 4️⃣ Dashboard (Home Screen)

### What You See on the Dashboard

*   System status (CPU, RAM, Disk)
*   Interface status (LAN, WAN)
*   Active users
*   Current threats
*   Web & application usage
*   VPN status
*   Alerts & warnings

🎯 **Purpose:**  
Quick health and security overview of your firewall

***

## 5️⃣ CONTROL Menu (Traffic Management)

### 📌 Firewall Rules and NAT

#### ➤ Firewall Rules

    CONTROL → Rules and Policies → Firewall Rules

Used to:

*   Allow / Deny traffic
*   Control source, destination, service
*   Apply security profiles

**Example:**  
Allow LAN users → Internet (HTTP/HTTPS)

***

#### ➤ NAT Rules

    CONTROL → Rules and Policies → NAT Rules

Used for:

*   Internet access
*   Port forwarding
*   Server publishing

**Example:**  
Public IP → Internal Web Server (Port 80)

***

#### ➤ SD‑WAN Rules

Used to:

*   Select best WAN link
*   Load balancing
*   Failover logic

***

## 6️⃣ PROTECT Menu (Security Features)

This menu focuses on **threat prevention**.

***

### 🔐 Web Protection

    PROTECT → Web

Features:

*   URL filtering
*   Category‑based blocking
*   HTTPS inspection
*   SafeSearch enforcement

**Example:**  
Block Adult & Gambling websites for employees

***

### 🦠 Malware Protection

    PROTECT → Malware

Features:

*   Antivirus scanning
*   Zero‑day threat protection
*   File type control

***

### 📱 Application Control

    PROTECT → Applications

Controls applications like:

*   Facebook
*   YouTube
*   Torrent
*   VPN apps

**Example:**  
Block Torrent, Allow YouTube with bandwidth limit

***

### 🛡 IPS (Intrusion Prevention System)

    PROTECT → Intrusion Prevention

Protects against:

*   Network attacks
*   Exploits
*   Port scanning

***

## 7️⃣ DETECT Menu (Monitoring & Threat Visibility)

### 🔍 Live Logs

    DETECT → Log Viewer

Real‑time logs:

*   Firewall traffic
*   Web access
*   Authentication
*   VPN activity

***

### 🚨 Alerts & Threats

Shows:

*   Security detections
*   Blocked attacks
*   Anomalies

***

## 8️⃣ CONFIGURE Menu (Network & Object Configuration)

***

### 🌐 Network Objects

    CONFIGURE → Network

Used to create:

*   Hosts
*   IP ranges
*   Subnets
*   MAC objects

***

### 🔌 Interfaces

    CONFIGURE → Network → Interfaces

Configure:

*   LAN, WAN, DMZ
*   IP addresses
*   Zones

***

### 👥 Authentication

    CONFIGURE → Authentication

Integrate:

*   Local users
*   Active Directory
*   LDAP
*   RADIUS

***

### 🎯 Services

    CONFIGURE → Services

Create custom ports:

*   TCP/UDP services
*   Service groups

***

## 9️⃣ SYSTEM Menu (Firewall Management)

***

### ⚙ System Services

    SYSTEM → Administration

Manage:

*   Admin access
*   Certificates
*   Password policies

***

### 🔄 Backup & Restore

    SYSTEM → Backup & Firmware

Used for:

*   Configuration backups
*   Firmware upgrades
*   Rollback

***

### 🔁 High Availability (HA)

    SYSTEM → High Availability

Firewall redundancy:

*   Active–Passive mode
*   Automatic failover

***

### 🌍 VPN Configuration

    SYSTEM → VPN

Types:

*   SSL VPN (remote users)
*   IPsec VPN (site‑to‑site)

***

## 🔟 REPORTS Menu (Visibility & Compliance)

### 📊 Reports and Analytics

    REPORTS → Reports

View reports on:

*   Web usage
*   Security threats
*   Application usage
*   User activity

📥 Can be downloaded as PDF or CSV

***

## 1️⃣1️⃣ Typical Admin Workflow (Real Scenario)

**Example: Internet Access for Employees**

1.  Create LAN Interface
2.  Configure WAN Interface
3.  Create Firewall Rule (LAN → WAN)
4.  Enable Web & Malware protection
5.  Apply Application Control
6.  Monitor logs & reports

***

## 1️⃣2️⃣ Best Practices for Beginners

✅ Always:

*   Use security profiles in rules
*   Backup before firmware upgrade
*   Monitor logs daily
*   Use user‑based policies

❌ Avoid:

*   Allow Any‑Any rules
*   Disabling IPS globally
*   Exposing management from WAN

***

## ✅ Final Summary (One‑Liner)

> **The Sophos Web Admin Console is a centralized, web‑based interface that allows administrators to control traffic, enforce security, monitor threats, and manage the entire firewall efficiently.**
