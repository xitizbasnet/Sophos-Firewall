# 🔐 How to Use Sophos VPN on Ubuntu Linux

## 📘 Overview

This document provides step-by-step instructions for configuring and connecting to the Sophos SSL VPN on Ubuntu Linux using OpenVPN.

---

# 📋 Prerequisites

Before starting, ensure the following:

* Ubuntu Linux system
* Internet connectivity
* Valid Sophos VPN user credentials
* Access to the Sophos Firewall Admin Portal
* OpenVPN package installation permission (`sudo` access)

---

# 🖥️ Step 1: Open Terminal

Open the terminal application on Ubuntu Linux.

---

# 🔄 Step 2: Update the Operating System

Run the following command to update the OS package repository:

```bash
sudo apt-get update
```

### 🔑 Authentication

When prompted:

```text
enter the password: *****
```

Press **Enter**.

---

# 📦 Step 3: Install OpenVPN

Install OpenVPN using the following command:

```bash
sudo apt-get install OpenVPN
```

---

# 👤 Step 4: Verify VPN User Permission in Sophos Firewall

> ⚠️ Note: Ensure the user has VPN access permission before proceeding.

## 🌐 Access Sophos Firewall

Open a browser and navigate to:

```text
172.16.16.16:444
```

### 🔐 Login Credentials

```text
username: admin
password: ****
```

Click **Login**.

---

## 🧭 Navigate to User VPN Settings

Go to:

```text
Configure > Authentication > Users > xitiz.basnet > VPN
```

Search for:

```text
user ssl vpn policy: office_SSL_VPN
```

> 📝 Note: `office_SSL_VPN` is the VPN policy that has been created.

Click **Save**.

---

# 📥 Step 5: Download Sophos VPN Configuration File

## 🌐 Open VPN Portal

Open Firefox and enter the Sophos VPN portal IP address:

```text
172.16.16.16:443
```

---

## ⚠️ SSL Warning Message

The browser may display:

```text
Your connection is not private
```

Proceed with the following steps:

1. Click **Advanced**
2. Click:

```text
Proceed to 172.16.16.16 (unsafe)
```

---

## 🔐 Login to VPN Portal

Enter the Sophos VPN user credentials:

```text
username: xitiz.basnet
password: ******
```

Click **Login**.

> ⚠️ Note: Make sure the user has VPN portal access permission enabled.

---

# 📁 Step 6: Download VPN Configuration File

Locate the VPN configuration setup file:

```text
SSL VPN Configuration
```

Choose:

```text
Download for Windows, macOS, Linux
```

Use with:

```text
Sophos Connect and OpenVPN Connect v2.2 clients
```

The file will be downloaded as:

```text
ssvpn-xitiz.basnet-client-config.ovpn
```

---

# 📂 Step 7: Move Configuration File

Move the downloaded file from the **Downloads** folder to the **Documents** folder for safe storage and future use.

Example:

```text
Downloads → Documents
```

---

# 🖥️ Step 8: Open Terminal in Documents Folder

Open the terminal inside the **Documents** directory where the `.ovpn` file is stored.

---

# 🚀 Step 9: Connect to Sophos VPN

Run the following command:

```bash
sudo OpenVPN ssvpn-xitiz.basnet-client-config.ovpn
```

Press **Enter**.

---

## 🔑 Enter System Password

When prompted:

```text
enter the computer password
```

Press **Enter**.

---

# 🔐 Step 10: Enter VPN Authentication Credentials

You will be prompted for VPN credentials:

```text
Enter Auth Username: xitiz.basnet
Enter Auth Password: *****
```

Press **Enter**.

---

# ⏳ Step 11: Wait for VPN Initialization

Wait for the VPN connection process to complete.

Successful connection message:

```text
Initialization Sequence Completed
```

> ✅ This confirms that the VPN connection is successfully established.

---

# 📡 Step 12: Verify VPN Connection from Sophos Firewall

To confirm whether the VPN is connected:

## 🧭 Navigate to:

```text
Monitor & Analyze > Current Activities > Live Users
```

You can verify connected VPN details such as:

```text
User ID: 8
Username: xitiz.basnet
Client Type: SSL VPN Tunnel
Host IP: 192.168.99.130
IP Version: IPv4
```

---

# ❌ Step 13: Disconnect VPN from Sophos Firewall Admin Portal

If required, the VPN can be disconnected directly from the Sophos Firewall portal.

## Steps:

1. Select the connected user using the tick mark
2. Click **Disconnect**
3. Click **Disconnect** again to confirm

A confirmation message will appear:

```text
VPN is disconnected
```

> ⚠️ Note: Only firewall administrators have permission to disconnect users from the Sophos portal.

---

# 🖥️ Step 14: VPN Behavior on Ubuntu Client

On the Ubuntu client machine:

* Closing the terminal window will automatically disconnect the VPN.
* To keep the VPN connected continuously:

  * Do **not** close the terminal
  * Only **minimize** the terminal window

---

# 📌 Important Notes

| ⚠️ Item            | Description                                        |
| ------------------ | -------------------------------------------------- |
| VPN Access         | Ensure the user has SSL VPN permission assigned    |
| VPN Portal Access  | User must have access to the VPN portal            |
| Terminal Session   | VPN disconnects automatically if terminal closes   |
| Configuration File | Store `.ovpn` file securely for future use         |
| Admin Rights       | Only firewall admins can disconnect users remotely |

# 🛠️ Troubleshooting Tips

| Issue                         | Possible Solution                               |
| ----------------------------- | ----------------------------------------------- |
| Authentication Failed         | Verify username and password                    |
| VPN Not Connecting            | Check internet connectivity and VPN permissions |
| `.ovpn` File Missing          | Re-download from Sophos VPN portal              |
| SSL Warning Appears           | Proceed manually after verifying firewall IP    |
| VPN Disconnects Automatically | Keep terminal window open                       |
