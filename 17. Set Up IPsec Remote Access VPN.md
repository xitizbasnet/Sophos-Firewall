# 🔐 How to Set Up IPsec Remote Access VPN

## 📘 Overview

Remote Access VPN has **2 types**:

* **IPsec VPN**
* **SSL VPN**

This document explains how to configure an **IPsec Remote Access VPN** on a firewall device and how to connect a client computer using the exported VPN configuration.

---

# 🧩 What is IPsec Remote Access VPN?

An **IPsec Remote Access VPN** allows remote users to securely connect to an office or enterprise network over the internet using encrypted tunnels.

The setup consists of **3 major parts**:

1. **Firewall Core Configuration**

   * Device Access
   * IP Host
   * Firewall Rule

2. **VPN Configuration**

   * Remote Access VPN → IPsec

3. **User-Level Configuration**

   * Export VPN Configuration
   * VPN Client Setup on Computer

---

# ⚙️ Part 1: Firewall Core Configuration

---

## 🔓 Step 1: Enable Device Access

Navigate to:

```text
System > Administration > Device Access
```

### Under the WAN Zone

Enable the following by ticking the checkboxes:

* ✅ VPN Portal
* ✅ IPsec

### Under the VPN Zone

Enable the following:

* ✅ Ping/Ping6
* ✅ DNS
* ✅ VPN Portal

### Under the DMZ Zone

Enable:

* ✅ VPN Portal

### Under the LAN Zone

Enable:

* ✅ VPN Portal

### Save Configuration

Click:

```text
Apply > OK
```

---

## 🌐 Step 2: Create an IP Host

Navigate to:

```text
System > Hosts and Services > IP Host
```

Click:

```text
Add
```

> ⚠️ **Important Note**
>
> Avoid using commonly used home networks such as:
>
> * `192.168.0.1`
> * `192.168.1.1`
>
> Also, do not use the same network range already used in the office network.

### Configure the Following Settings

| Setting     | Value               |
| ----------- | ------------------- |
| Name        | FOR VPN             |
| Description | FOR VPN             |
| IP Version  | IPv4                |
| Type        | Network             |
| IP Address  | 192.168.99.0        |
| Subnet      | /24 (255.255.255.0) |

Click:

```text
Save
```

---

# 🔥 Step 3: Create Firewall Rule

Navigate to:

```text
Protect > Rules and Policies > Firewall Rules
```

Click:

```text
Add Firewall Rule > New Firewall Rule
```

---

## 🛡️ Firewall Rule Configuration

### General Settings

| Setting       | Value     |
| ------------- | --------- |
| Rule Name     | ipsec_VPN |
| Rule Position | Top       |
| Action        | Accept    |
| Rule Group    | None      |

Enable:

* ✅ Log Firewall Traffic

---

## 📥 Source Settings

| Setting                     | Value        |
| --------------------------- | ------------ |
| Source Zone                 | VPN          |
| Source Networks and Devices | FOR VPN      |
| During Scheduled Time       | All the Time |

---

## 📤 Destination and Service Settings

| Setting              | Value    |
| -------------------- | -------- |
| Destination Zone     | DMZ, LAN |
| Destination Networks | Any      |
| Services             | Any      |

Click:

```text
Save
```

---

# 🔐 Part 2: VPN Configuration

## Configure IPsec Remote Access VPN

Navigate to:

```text
Configure > Remote Access VPN > IPsec
```

---

## ⚙️ General Settings

Enable and configure the following:

| Setting             | Value                |
| ------------------- | -------------------- |
| IPsec Remote Access | ✅ Enabled            |
| Interface           | port2 - 136.36.36.36 |
| IPsec Profile       | defaultremoteaccess  |
| Authentication Type | Preshared Key        |
| Preshared Key       | abc123456            |
| Local ID            | Leave Blank          |
| Remote ID           | Leave Blank          |

### Allowed Users and Groups

Add users as required:

```text
xitiz.basnet
basnet.xitiz
```

> 📝 **Note**
>
> Generally, offices have multiple ISP connections.
>
> Choose the ISP interface according to the requirement so that:
>
> * VPN traffic uses the selected ISP public IP
> * Routing occurs through the selected ISP
> * Logging is maintained properly
> * Other ISP traffic remains unaffected

---

## 👤 Client Information

| Setting        | Value                          |
| -------------- | ------------------------------ |
| Name           | headoffice                     |
| Assign IP From | 192.168.99.10 - 192.168.99.100 |
| DNS Server 1   | 8.8.8.8                        |
| DNS Server 2   | 4.4.2.2                        |

Do **NOT** enable:

* ❌ Allow leasing IP address from RADIUS server for L2TP, PPTP, and IPsec remote access

---

## ⏱️ Idle Time Settings

| Setting                        | Value    |
| ------------------------------ | -------- |
| Disconnect When Tunnel is Idle | ✅ Enable |
| Idle Session Time Interval     | 240      |

---

## ⚡ Advanced Settings

Enable the following:

* ✅ Use as Default Gateway
* ✅ Allow Users to Save Username and Password

Click:

```text
Apply > OK
```

---

# 💻 Part 3: Export Configuration and Install VPN Client

---

## 📦 Export VPN Configuration

Navigate to:

```text
Configure > Remote Access VPN
```

At the bottom of the page, click:

```text
Export Connection
```

Download the configuration file.

---

## ⬇️ Download VPN Client

From the top menu, click:

```text
Download Client
```

Download the VPN client installer.

---

# 🛠️ Install Sophos Connect Client

After downloading:

1. Go to the **Downloads** folder.
2. Extract both downloaded files using **WinRAR**.

---

## 📂 Open the Following Folder

```text
sophosconnect_installer > sophosconnect_2.5.0_GA(ipsec_and_SSLVPN)
```

Run the installer.

---

## 🧾 Installation Steps

1. Tick:

```text
I accept the terms
```

2. Click:

```text
Next
```

3. Tick:

```text
Launch Sophos Connect when installer closes
```

4. Click:

```text
Finish
```

---

# 🔗 Import VPN Connection

After installation:

1. Go to the **taskbar** (bottom-right corner).
2. Click the **Sophos Connect** icon.
3. Click:

```text
Import Connection
```

4. Select the file:

```text
headoffice (.SCX)
```

> 📝 **Note**
>
> You may see two files:
>
> * `headoffice (.SCX)`
> * `headoffice (.TGB)`
>
> Select the `.SCX` file.

---

# 🔐 Connect to VPN

Click:

```text
Connect
```

A login popup will appear.

Enter the following credentials:

| Field    | Value        |
| -------- | ------------ |
| Username | xitiz.basnet |
| Password | ********     |

Enable:

* ✅ Remember Username and Password

Click:

```text
Sign In
```

# ✅ VPN Connection Successful

After authentication, the VPN will connect successfully.

* Use strong preshared keys.
* Avoid overlapping IP ranges.
* Regularly review firewall logs.
* Limit VPN access to authorized users only.
* Use secure DNS servers.
* Keep Sophos Connect updated.
